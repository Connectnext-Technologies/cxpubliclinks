URL Extraction Test Page

This page contains various types of links to test the URL extraction functionality.

Internal Links (Same Domain)

  • Simple relative link
  • Relative link with dot notation
  • Parent directory relative link
  • Multiple parent directory relative link
  • Failure Multiple parent directory relative link
  • Absolute path link
  • HTML with .htm extension
  • Relative link with query string
  • Full URL to same domain
  • Full URL with complex query string

External Links (Different Domains)

  • External site with HTTPS
  • External site with HTTP
  • External site with subdomain
  • External link with query parameters

Edge Cases

  • JavaScript protocol link
  • Hash/anchor link
  • Empty href attribute
  • Mailto link
  • Tel link
  • URL with spaces
  • Non-HTML document
  • Text with embedded URL like https://embedded-url.com/page.html but not in an anchor tag
  • Link with the same text as href

Malformed Links

  • Link with quote in URL
  • Link with bracket in URL
  • Link with incomplete protocol
  • Absolute path without domain
